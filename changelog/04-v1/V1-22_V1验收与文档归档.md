## 文档归档

### 1. 更新 AGENTS.md / CLAUDE.md

反映 V1 架构变更：
- `Node.Labels []string` 替代 `Node.Label string`
- `ConnectorFactory` 模式替代硬编码
- `SnapshotDiff` 新增 `ChangedNodes`/`ChangedRels`
- `EntityTypeSpec.Extends` 继承机制
- `AuditLog` 审计日志
- `MetaCache` 缓存层

### 2. 记录 V2 技术债

| 债务项 | V1 状态 | V2 升级方案 |
|--------|---------|------------|
| `Connector.Stream()` | 仍返回 `ErrNotImplemented` | Kafka consumer 适配 |
| 内存 Channel 缓冲 | 保持现状 | Kafka 持久化 |
| 无 HTTP REST API | 仅 MCP | Gin HTTP API |
| 无分析引擎 | 无 | Impact/RCA/Simulation Engine |
| 无 PostgreSQL 元数据 | YAML + 内存 | PostgreSQL 持久化 |
| AuditLog 内存存储 | FIFO 淘汰 | PostgreSQL 持久化 |
| 单 Neo4j 实例 | CE `_db` 属性隔离 | Enterprise 真多 DB |
| 语义映射规则外置 | 内嵌在 Schema | 拆分 `mapping_rules.yaml` |
| golangci-lint CI 集成 | 本地手动运行 | GitHub Actions 自动化 |
